# CardioRisk Prediction

## Project Overview
This project predicts cardiovascular disease risk from patient health records using exploratory data analysis, feature engineering, and machine learning models.

## Files
- `CardioRisk_Prediction.ipynb`: Main notebook for cleaning, EDA, modelling, and evaluation.
- `data/health_data.csv`: Source dataset used by the notebook.

## What This Project Covers
- Data loading and cleaning
- Exploratory data analysis
- Feature engineering, including BMI creation
- Model training and comparison

## How to Run
1. Create and activate a Python environment.
2. Install core libraries: pandas, numpy, matplotlib, seaborn, scikit-learn, jupyter.
3. Open `CardioRisk_Prediction.ipynb` and run cells from top to bottom.

## Notes
- The notebook reads the dataset from `data/health_data.csv`.

## Data Availability
This project includes the notebook and dataset needed to reproduce the analysis in the repository.
